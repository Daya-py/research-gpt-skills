# AI Lexicon

## Purpose

This document defines words, phrases, sentence structures, and stylistic patterns commonly associated with AI-generated writing.

The Academic Humanizer uses this lexicon as a reference during the audit phase. Entries are not absolute replacement rules. Each entry must be evaluated in context before revision.
