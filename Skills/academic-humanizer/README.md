# Academic Humanizer

A professional ChatGPT Skill for transforming AI-assisted academic and technical writing into natural, publication-quality prose while preserving scientific accuracy, technical meaning, citations, equations, and domain-specific terminology.

## Features

- Context-aware editing
- AI-associated writing pattern detection
- Technical terminology preservation
- Citation integrity verification
- Publication-quality rewriting
- Discipline-aware editing
- Minimal-edit philosophy
- Integrated AI lexicon

## Supported Documents

- Journal manuscripts
- Conference papers
- Theses and dissertations
- Research proposals
- Technical reports
- Engineering documentation
- Scientific articles
- Literature reviews

## Installation

1. Clone or download this repository.
2. Copy `SKILL.md` into your ChatGPT Skill or Custom GPT environment.
3. Keep the `references/` directory alongside the skill for maintenance and future expansion.
4. Use the examples and tests to validate changes before updating the skill.

## Design Philosophy

This project is built on four principles:

- Preserve meaning.
- Preserve technical accuracy.
- Eliminate AI-associated writing patterns where appropriate.
- Produce publication-quality academic writing.

This skill prioritizes scientific integrity over aggressive rewriting.
